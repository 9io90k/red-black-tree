# Красно-черное дерево

Реализация самобалансирующегося двоичного дерева поиска (Red-Black Tree) на C++.
Проект поддерживает базовые операции вставки, удаления и поиска, а также корректное восстановление свойств дерева после модификаций.

---

## Свойства красно-черного дерева

- Корень всегда черный 
- Листья (NULL-узлы) считаются чёрными  
- У красного узла оба ребёнка — чёрные (нет двух красных подряд)  
- Для любого узла количество черных узлов на пути до любого листа одинаково (black-height)
- Узел либо черный, либо красный

---

## Возможности

- Вставка узла с восстановлением свойств (fixViolation)  
- Удаление узла с корректировкой структуры (fixDelete)  
- Поиск узла по ключу  
- Поиск минимального элемента  
- Левый и правый повороты  
- Полное ручное управление памятью  
- Рекурсивное удаление дерева в деструкторе

---

## Сложность операций

| Операция | Сложность |
|----------|-----------|
| Поворот | O(1) |
| Вставка | O(log n) |
| Удаление | O(log n) |
| Поиск | O(log n) |
|fixViolation и fixDelete | O(log n) |
